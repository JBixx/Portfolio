# Portfolio — Hirwa Jean Baptiste

Salut. Voici mon portfolio perso.

Je m’appelle **Hirwa Jean Baptiste**, développeur full-stack basé à Brazzaville. Ce site présente mon parcours, mes compétences et une sélection de projets sur lesquels j’ai travaillé (web, IA, IoT, automatisation).

Rien de trop fancy côté stack : du HTML, du CSS et un peu de JavaScript vanilla. L’idée, c’était de garder quelque chose de léger, lisible et facile à déployer.

---

## Ce que tu trouveras sur le site

- Une page d’accueil avec mon profil et un aperçu rapide
- Une section À propos (parcours, CFI-CIRAS, distinctions)
- Des compétences avec pourcentages
- Une galerie de projets (6 projets, avec tags et liens)
- Une timeline d’expérience
- Un formulaire de contact (avec motif)
- Une newsletter
- Un mode sombre
- Une navigation mobile avec menu hamburger

---

## Techno

| Élément | Choix |
|--------|--------|
| Markup | HTML5 |
| Styles | CSS3 (variables, grid, flexbox) |
| Script | JavaScript (vanilla) |
| Icônes | Font Awesome 6 |
| Fonts | Outfit + Plus Jakarta Sans |

Pas de framework. Pas de bundler. Tu ouvres `index.html` et ça tourne.

---

## Lancer en local

1. Clone le repo :

```bash
git clone https://github.com/JBixx/Portfolio.git
cd Portfolio
```

2. Ouvre `index.html` dans ton navigateur  
   (ou sers le dossier avec Live Server / un petit serveur local).

## Fonctionnalités un peu plus “pro”

- **Mode sombre** : toggle dans la navbar, préférence sauvegardée dans `localStorage`
- **Menu hamburger** : drawer mobile avec overlay
- **Animations au scroll** : Intersection Observer (respecte `prefers-reduced-motion`)
- **Newsletter** : validation email côté client
- **Responsive** : desktop, tablette, mobile
